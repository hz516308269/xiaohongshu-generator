# 小红书文案生成器

## 项目说明

这是一个小红书文案生成工具，用户可以选择文案类型、输入关键词，一键生成适合小红书的文案。

## 文件结构

D:\钱多多创业\
├── index.html   # 主页面
├── style.css    # 样式文件
├── script.js    # 逻辑脚本
└── README.md    # 说明文件

## 功能特点

1. 多类型支持：美妆、美食、旅行、穿搭、生活、学习、健身、好物推荐
2. 多风格选择：日常分享、专业测评、情感共鸣、幽默搞笑、简洁明了
3. 字数可控：简短、适中、详细
4. 标签生成：自动生成相关话题标签
5. 一键复制：方便直接使用
